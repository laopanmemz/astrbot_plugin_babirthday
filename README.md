# AstrBot 蔚蓝档案学生生日提醒

这是一个基于 AstrBot 的，实现Blue Archive学员生日提醒的插件。

### 使用方法

---

· 命令：

`/ba数据更新` 手动更新学生数据

`/ba生日`        手动获取**当天**学生生日情况

`/ba本周生日` 获取**本周**学生生日情况

<br>

· 配置：

1. Github 加速地址 [列表]

   ba学生数据[从Github的SchaleDB仓库](https://github.com/SchaleDB/SchaleDB)拉取，但由于境内网络问题可能会clone失败，插件配置内已经预置了加速地址，如果无法使用可以自行修改加速地址。

2. 是否带学生图片 [布尔值]

   开启状态时，发送当天/本周学生生日时带上学生的图片。

3. 每日提醒时间 [文本]

   填写每日提醒时间，格式为 HH:MM ，例如：8:00，13:00 等，冒号要用英文的！
